Hello world. This is my very first +300 lines project. I've been studying Python and this is the result of months of practicing.

I've created this bank system using Python solely. I have created the following features described below as they appear in the program.

### User functions:
- Transaction info generator (works internally, getting all the information of every process, either successful or not)
- Deposit
- Transfer
- Show account balances
- Freeze and unfreeze accounts
- Withdraw functions (adapted to either checking or savings accounts)

### Bank functions:
- Create accounts (saving and checking)
- Find account using user ID
- Delete accounts
- Transfer between accounts
- Show all accounts in the system

### Functions definition:
- validate_name()
- validate_amount()
- Find_account()
- authenticate_pin()

I created these helper functions to make data verification easier, improve code readability, and prevent invalid user input.


### Menu system:
A console-based menu system that allows users to interact with the banking application.

I want to share that I'm continuously improving the program. Future changes are meant to streamline blocks of code by defining functions that validate whether accounts exist or not.
